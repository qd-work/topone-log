## 第 1 步：询盘能收到（接通飞书）⭐ 最重要

现在网站是"模拟模式"：海外货代点 "Get a Quote" 提交询盘，只在服务器日志里，**你这边收不到**。接通飞书后，每条询盘会自动：① 写进飞书多维表格 ② 发邮件通知 ③ 推送到飞书群。

**【你准备】（二选一）**：
- **方式 A（最省事，推荐）**：把飞书管理员账号给 Vincent，我全程建好应用 + 表格 + 机器人 + 接通。
- **方式 B（你自己建）**：在飞书管理后台做 4 件事（每件飞书都有引导，不复杂）：
  1. 建一个「企业自建应用」→ 拿到 **App ID + App Secret**
  2. 给这个应用开通「多维表格」读写权限
  3. 建一个多维表格（用来登记询盘）→ 拿到 **多维表格的 Token**（表格网址里的那段字符）+ **数据表 ID**
  4. 建一个飞书群，在群里加一个「自定义机器人」→ 拿到 **Webhook 地址**（+ 签名校验密钥，可选）
- 另外给我一个**发件邮箱**（飞书企业邮箱，或腾讯/阿里企业邮箱）：邮箱地址 + 客户端授权码（用来发"新询盘"邮件通知）。
- 告诉我**接收通知的邮箱**（你 / 销售查收询盘的邮箱）。

**【我来配】**：把以上 7-8 个凭证填进网站配置，接通 3 路通知。

> 效果：客户一提交询盘，你飞书群立刻弹通知 + 多维表格自动登记一行，销售马上能跟进，不漏单。

**【你决策 + 准备】**：

**【我来配】**：域名解析 + SSL（HTTPS，部署平台自动配）。
